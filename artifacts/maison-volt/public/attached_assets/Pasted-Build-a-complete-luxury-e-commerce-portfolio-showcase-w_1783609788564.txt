Build a complete luxury e-commerce portfolio showcase website in one single `index.html` file.

Project name: Maison Volt

Goal:
Create a premium e-commerce website that feels like a real high-end online store. The site should focus on luxury shopping, clean product presentation, smooth interactions, and a discreet product-level concierge chat feature. The concierge should be available from each product, but the website should not feel like an AI-first website.

Technical requirements:
- Use only HTML, CSS, and vanilla JavaScript.
- Everything must be inside one `index.html` file.
- No external frameworks.
- No backend.
- No real payment system.
- No real AI API.
- Use mock product data and scripted JavaScript responses.
- The website must run directly in the browser.

Design direction:
- Luxury dark editorial e-commerce style.
- Sharp corners, not rounded.
- Minimal, premium, polished layout.
- Use thin borders, refined spacing, elegant typography, and smooth animations.
- Avoid playful visuals.
- Avoid emoji.
- Avoid bubbly UI.
- Avoid loud neon effects.
- Make it feel like a luxury fashion or premium technology store.

Color palette:
- Background: #080808

Typography:

Chat logic:
- If the message includes “shipping” or “delivery”, answer with shipping policy.
- If the message includes “return” or “refund”, answer with return policy.
- If the message includes “warranty”, answer with warranty information.
- If the message includes “track” or an order ID, return a mock order status.
- If the message includes “recommend”, suggest products based on the user’s need.
- If the message mentions two products, compare them.
- If a product context exists, answer based on that product’s specs.

Mock policies:
- Standard delivery: 3–5 business days.
- Express delivery: 1–2 business days.
- Returns: accepted within 7 days for damaged, defective, or incorrect items.
- Warranty: 1-year limited warranty for all products.
- Mock order examples:
  - ORD-2026-4921: Out for delivery
  - ORD-2026-1834: Processing
  - ORD-2026-7750: Delivered

Animation requirements:
- Smooth scroll.
- Fade-in sections on scroll.
- Product card reveal animation.
- Slow image zoom on hover.
- Thin border reveal on hover.
- Cart drawer slide-in.
- Chat panel slide-in.
- Product modal fade and scale.
- Button underline animation.
- Subtle hero parallax.
- Animations should feel refined and expensive, not playful.

Responsive requirements:
- Desktop, tablet, and mobile support.
- Product grid should adapt from 4 columns to 2 columns to 1 column.
- Cart drawer and chat panel should work well on mobile.

Important:
- Do not use emoji anywhere.
- Do not use rounded bubbly cards.
- Do not make the website look like a chatbot demo.
- Prioritize e-commerce luxury experience.
- Make the final code clean, readable, and presentation-ready.
